# PopX React Assignment

A React + Vite implementation of the provided PopX mobile UI, centered inside a desktop webpage with working page-to-page navigation.

## Pages

- Welcome
- Login
- Register
- Account Settings

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Notes

- Built with React, Vite, and React Router.
- The account screen uses a local SVG avatar so the UI does not depend on an external image service.
- Form state is stored locally to keep the demo flow working across navigation.
